# 术语表

| 术语                | 含义                                                                                                              |
| ------------------- | ----------------------------------------------------------------------------------------------------------------- |
| SRVF                | 深圳公益救援队项目业务域                                                                                          |
| 当前上传基准        | 本轮上传的 `srvf-admin-web_git-main_c2001c9_20260705.zip`                                                         |
| 用户确认代码包      | 用户明确反馈测试通过的业务代码包，本轮记录为 `srvf-admin-web_v7.8.0_scoped-authz-browser-smoke_full_20260703.zip` |
| docs-only 包        | 只改文档，不改变业务代码行为的包                                                                                  |
| 活动轴              | 以活动详情为上下文，管理报名、考勤等子资源                                                                        |
| 队员轴              | 以队员 360 为上下文，管理档案、证书、归属、任职等子资源                                                           |
| 组织轴              | 以组织树节点为上下文，管理组织成员、在任职务等                                                                    |
| scoped-authz        | 后端按角色、组织、资源范围等综合判权的权限体系                                                                    |
| action-state batch  | 后端合并权限与状态机，返回按钮可用性的批量接口                                                                    |
| `/get-async-routes` | pure-admin mock 动态路由接口，不是 SRVF 后端接口                                                                  |
