# 设计系统

## 一、基础

本项目继承 Pure Admin / Element Plus 后台设计体系。当前不单独引入品牌视觉重构。

## 二、组件优先级

1. 现有 SRVF 页面中的模式。
2. starter 中已有 `Re*` 组件。
3. 大参考包中的 UI 结构和交互范式。
4. 自定义组件。

## 三、禁止

- 不因参考包有某页面就新增业务模块。
- 不复制参考包 mock、接口、RBAC、tenant。
- 不新增重依赖。

## 四、后续建议

完成 P0/P1 后，可以做一次 SRVF 后台 UI 一致性清单：表格密度、筛选区、弹窗宽度、Drawer 详情、空状态、权限禁用态、状态标签色。
